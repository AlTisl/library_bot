# library_bot
Телеграм бот бібліотеки Національного університета "Запорізька політехніка". Надає користувачам доступ до онлайн-послуг бібліотеки (пошук літератури, оформлення заявок і т. д.), містить вимоги до оформлення публікацій та інші корисні матеріали.

## Реалізація
Бот написаний на Python із використанням бібліотеки aiogram 3.20.0. Для зберігання станів, а також для зберігання FileID файлів, що завантажуються, використовується Redis.

## Файли та директорії
- requirements.txt - список залежностей проєкта;
- main.py - точка входу в застосунок;
- config_reader.py - містить класс Settings, який зчитує з файлу .env болт-токен і параметри підключення до сховища Redis;
- database.py - містить функції для запису до сховища Redis та читання звідти значень FileID статичних файлів;
- media_sending.py - функції для відправки статичних файлів (зображень, анімації, документів) або за FileID (якщо він вже є), або шлхом безпосереднього завантаження файлу;
- decorators.py - містить декоратор для функцій з media_sending, який перевіряє, чи існує у сховищі FileID об'єкта, що завантажується, і залежно від цього передає в обернену функцію або об'єкт файла, або його FileID;
- keyboards.py - клавіатури та інлайн кнопки;
- strings.py - текстовий вміст повідомлень у вигляді об'єктів класу Text;
- handlers/ - обробники команд;
- constants/ - незмінні об'єкти: шляхи до статичних файлів, назви кнопок, url-адреси для інлайн-кнопок;
- resources/ - статичні файли, що використовуються в проєкті.
